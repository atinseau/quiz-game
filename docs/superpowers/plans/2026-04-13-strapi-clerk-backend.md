# Strapi + Clerk Backend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Strapi v5 backend with Clerk JWT authentication and on-demand Player sync to the quiz-app monorepo.

**Architecture:** Strapi v5 in `apps/backend/` (port 1337). A global middleware intercepts Bearer tokens, verifies them with `@clerk/backend`, and upserts a Player content-type. The React client wraps in `<ClerkProvider>`, sends Clerk JWTs in API requests.

**Tech Stack:** Strapi v5, @clerk/backend, @clerk/clerk-react, Bun, React, TypeScript, SQLite (dev)

### Task 3: Create Clerk auth middleware

**Files:**
- Create: `apps/backend/src/middlewares/clerk-auth.ts`
- Modify: `apps/backend/config/middlewares.ts`

- [ ] **Step 1: Write the Clerk auth middleware**

Create `apps/backend/src/middlewares/clerk-auth.ts`:

```ts
import { verifyToken } from '@clerk/backend';

export default (config, { strapi }) => {
  return async (ctx, next) => {
    const authHeader = ctx.request.header.authorization;

    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      // No token — let the request through (public route or Strapi default auth)
      return next();
    }

    const token = authHeader.replace('Bearer ', '');

    try {
      const verified = await verifyToken(token, {
        secretKey: process.env.CLERK_SECRET_KEY,
      });

      const clerkId = verified.sub;
      const email = verified.email as string | undefined;
      const username = verified.username as string | undefined
        || verified.email as string | undefined
        || clerkId;

      // Find or create Player
      const existing = await strapi.documents('api::player.player').findFirst({
        filters: { clerkId },
      });

      if (existing) {
        ctx.state.player = existing;
      } else {
        const created = await strapi.documents('api::player.player').create({
          data: {
            clerkId,
            email: email || `${clerkId}@clerk.user`,
            username: username || clerkId,
          },
        });
        ctx.state.player = created;
      }
    } catch (error) {
      strapi.log.error('Clerk token verification failed:', error);
      ctx.status = 401;
      ctx.body = { error: 'Invalid or expired token' };
      return;
    }

    return next();
  };
};
```

Create `apps/client/src/lib/api.ts`:

```ts
const API_URL = "http://localhost:1337/api";

export async function apiFetch(
  path: string,
  getToken: () => Promise<string | null>,
  options: RequestInit = {}
) {
  const token = await getToken();
  const headers: Record<string, string> = {
    "Content-Type": "application/json",
    ...((options.headers as Record<string, string>) || {}),
  };

  if (token) {
    headers["Authorization"] = `Bearer ${token}`;
  }

  const res = await fetch(`${API_URL}${path}`, {
    ...options,
    headers,
  });

  if (!res.ok) {
    throw new Error(`API error: ${res.status} ${res.statusText}`);
  }

  return res.json();
}
```
